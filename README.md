# Obatin Yuk

Aplikasi Pengingat Minum Obat berbasis Android Studio.

## Fitur

- Splash Screen
- Login
- Home
- Tambah Jadwal Obat
- Riwayat Obat
- Notifikasi Pengingat
- Profil Pengguna

## Teknologi

- Java
- Android Studio
- SQLite
- Material Design
